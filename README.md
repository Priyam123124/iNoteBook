# 📒 iNoteBook – Notes Web Application

A full-stack notes application that allows users to securely create, manage, and organize their personal notes. Built using the MERN stack, this app provides authentication and real-time CRUD operations for efficient note management.

---

## 🚀 Features

- 🔐 User Authentication (Signup/Login)
- 📝 Create, Edit, and Delete Notes
- 📂 Organized note management
- 🔒 Secure routes using JWT authentication
- ⚡ Fast and responsive UI
- 🌐 RESTful API integration

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)
